# Albums API

A lightweight REST API service for managing a collection of music albums, built with Go and the Gin framework. The service provides basic CRUD operations with in-memory storage.

## Overview

This service implements a simple HTTP API that exposes endpoints for retrieving and managing album records. It's designed as a minimal example of a production-ready Go web service, demonstrating proper request handling, JSON serialization, and HTTP status code management.

## Requirements

- Go 1.16 or later
- Gin framework (automatically installed via go.mod)
